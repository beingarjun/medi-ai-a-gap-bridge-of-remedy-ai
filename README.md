## 🚦 Getting Started
1. 📝 `cp .env.example .env` and fill in credentials
2. 📦 `npm install` in both `server` and `web`
3. 🖥️ Start backend: `npm --workspace server run dev`
4. 💻 Start frontend: `npm --workspace web run dev`
